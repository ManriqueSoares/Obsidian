# Distâncias Elétricas — Conceitos

Relacionado: [[parte-ativa]] | [[comutadores]] | [[canecos]]

## O que são distâncias elétricas?

Distâncias mínimas de isolamento entre elementos de um transformador que garantem suportabilidade dielétrica sob tensões de ensaio (BIL, SIL, tensão aplicada) e em serviço.

## Distâncias principais no projeto

| Símbolo | Descrição |
|---|---|
| `d_AT_tanque` | Parte ativa (AT) → parede do tanque |
| `d_BT_tanque` | Enrolamento BT → parede do tanque |
| `d_AT_BT` | Entre enrolamentos AT e BT (cilindro de isolação) |
| `d_comut_AT` | Comutador → enrolamento AT |
| `d_comut_tanque` | Comutador → tanque |
| `d_entre_canecos` | Entre seções de bobina (canecos) verticalmente |
| `d_entre_fases` | Entre fases (transformador trifásico) |
| `d_fundo` | Parte ativa → fundo do tanque |
| `d_topo` | Parte ativa → tampa do tanque |

## Fatores determinantes

- Tensão máxima do equipamento `Um` (kV)
- Nível de isolamento (BIL em kVp / SIL em kVp / tensão aplicada em kVef)
- Meio isolante (óleo mineral, óleo vegetal, seco)
- Pressão e temperatura do óleo
- Geometria dos eletrodos

## Normas de referência

- **IEC 60076-3** — Transformadores de potência: Nível de isolamento, ensaios dielétricos e distâncias no ar
- **ABNT NBR 5356-3** — equivalente nacional
- **IEC 60076-1** — Requisitos gerais
